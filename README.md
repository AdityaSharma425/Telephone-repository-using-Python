# Telephone-repository-using-Python

This is a telephone repository project which can create a new telephone directory, add a new contact, display the directory, search for a contact by number as well as by the name.
csv Module is used in this project by performing file handling operations.
Place this file in  python directory and then run it.
But, first press choice 1 and it will create a new csv file for your directory.
You're good to go.
